# 四川小自考工具库

面向四川自考新手的信息库和工具箱，用来集中查看学校、专业、费用、大小自考区别、拿证时间、复习计划、论文和毕业学位事项。

## 当前数据覆盖

| 数据项 | 数量 |
| --- | ---: |
| 学校 | 51 所 |
| 院校专业 | 666 个 |
| 专科院校专业 | 172 个 |
| 专升本院校专业 | 494 个 |

## 功能清单

| 功能点名称 | 对应页面 | 细节内容 |
| --- | --- | --- |
| 首页总览 | `/` | 汇总四川小自考核心入口：查学校、查专业、查费用、大小自考、拿证时间、复习计划、论文工具；帮助用户先确定下一步去哪看。 |
| 查学校 | `/schools`、`/schools/:id` | 按学校查看应用型自考专业、专科/专升本层次、专业代码、助学价格、论文费用、课程数量、学校官网、核实状态。 |
| 查专业 | `/majors`、`/majors/:id` | 按专业名称、层次、学校筛选；查看同一专业有哪些学校开设；进入详情后看课程结构、费用、专业代码、难度评估。 |
| 查费用 | `/prices` | 单独整理助学价格、论文费用、费用待核对项，方便报名前做预算对比。 |
| 大小自考对比 | `/exam-types` | 解释大自考和小自考区别、适合人群、考试节奏、费用差异、报名选择思路。 |
| 拿证时间计算器 | `/certificate-time` | 按自考路径、专科/专升本、课程数量、统考/省考批次、通过预期、实践/论文状态估算拿证周期。支持按系统内学校专业快速带入。 |
| 论文阶段风险预留 | `/certificate-time` | 小自考 + 专升本 + 论文阶段时，可选择“一次通过”或“预留一次重做或补答辩”，用于更保守估算毕业时间。 |
| 专业方向时间画像 | `/certificate-time` | 针对医药护理、工程交通、农学园林、艺术传媒、财经管理、教育语言等方向，给出额外时间风险提醒。 |
| 复习计划生成器 | `/study-plan` | 按学校专业带入统考/省考课程，选择本次备考课程、剩余天数、每天学习时间、基础水平，生成复习小时分配、阶段计划和执行建议。 |
| 课程强度识别 | `/study-plan` | 自动识别数学、英语、技术、工程、财经、医药、食品、农学、法务、管理、体育、艺术等课程类型，并影响复习强度。 |
| 统考时段冲突提醒 | `/study-plan` | 统考课程会匹配内置考试时段，若选择课程存在同一场次冲突，会提示不能同时报考。 |
| 专业难度评估 | `/majors/:id`、`/schools/:id` | 基于课程表评估难度：数学压力、英语安排、实践/论文数量、统考门数、专业方向负担，并输出标签和建议。 |
| 新增专业难度判断 | `/majors/:id`、`/schools/:id` | 已覆盖食品、农学、艺术传媒、法学公安、管理类、教育语言、体育、工程技术、医药护理等新增方向。 |
| 流程库 | `/process` | 从报名前决策、选学校专业、注册报考、备考、论文毕业到避坑提醒，按完整路线图展示。 |
| 论文工具库 | `/thesis` | 面向小自考专升本论文，覆盖论文申请、选题、提纲、格式、查重、答辩、成绩归档。 |
| 论文规划器 | `/thesis` | 可按学校专业带入论文信息，辅助判断论文方向、题目风险、目录结构、材料准备和答辩重点。 |
| 毕业 / 学位指南 | `/graduation` | 说明毕业申请、论文实践、学位证条件、材料优先级；可按学校专业查看毕业和学位重点。 |
| 资料清单 | `/resources` | 按复习、论文、毕业、学位阶段整理资料方向，引导用户先判断阶段，再准备对应材料。 |
| 轻咨询入口 | 多页面 | 在首页、流程、资料、毕业、拿证时间等页面引导用户加微信咨询，用于学校、专业、路径、费用、拿证节奏等转化。 |
| 页面品牌与导航 | 全站 | 使用“四川小自考工具库”品牌名，顶部导航覆盖主要查询和工具页面，页面标签图标使用“自”字图标。 |

## 内容边界

本站内容用于信息整理和辅助决策，不替代官方通知，也不构成报名承诺、毕业承诺或学位承诺。学校、专业、费用、课程计划、毕业申请和学位条件等，请以四川省教育考试院、主考院校及学校当期通知为准。
