## 3. 下一步核验顺序

1. 专业名单：逐校核对官方一览表图片里的专科、专升本招生专业。
2. 专业代码与课程计划：对照四川省教育考试院 2025 年版专业考试计划。
3. 学费/助学费：逐校对照继续教育学院官网、招生简章或官方通知。
4. 论文/实践费用：逐校对照学校官网或继续教育学院通知；无法确认的继续标记为“待核对”。
5. 页面展示：不要统一写“全部官网已核”，应按字段显示“学校名单已核 / 专业待核 / 费用待核 / 论文费待核”。
